# Pricing Axes

## Three axes

- Packaging answers what is included in each plan.
- Value metric answers what the customer is charged for as usage or scale grows.
- Price point answers how expensive each package or usage band is.

## Why it matters

- Many pricing debates get muddled because the team is actually changing more than one axis at once.
- A strong pricing memo names the primary axis change so the recommendation is easier to test and defend.
- If the recommendation changes two axes at once, say which one is the real driver.
