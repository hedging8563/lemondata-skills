# Usage Notes

This reference file supports `devops-incident-runbook`.

## When this skill is most useful

- A runbook that can be followed during an active incident.
- Clear severity or impact framing and escalation paths.
- Mitigation, verification, and rollback or recovery guidance.

## Delivery hints

- The best incident runbooks are executable by someone who did not write them.
- Detection and triage should narrow the problem fast.
- Escalation paths belong in the runbook because incidents rarely stay contained.
