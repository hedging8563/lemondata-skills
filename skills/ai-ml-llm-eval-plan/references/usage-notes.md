# Usage Notes

This reference file supports `ai-ml-llm-eval-plan`.

## When this skill is most useful

- A clear evaluation objective tied to an LLM behavior or product outcome.
- A compact rubric with dimensions and grading logic.
- A test set or experiment plan with pass bars and next iteration guidance.

## Delivery hints

- A strong eval plan is reusable, not only thoughtful.
- Pass bars matter because they prevent endless subjective debate.
- The test set should reflect the real failure modes the product cares about.
