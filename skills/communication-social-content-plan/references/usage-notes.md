# Usage Notes

This reference file supports `communication-social-content-plan`.

## When this skill is most useful

- A platform-aware content plan tied to one goal and one audience.
- A small set of hooks or post angles that vary by platform.
- Drafts or outlines for multiple posts rather than one isolated message.
- A lightweight distribution pack that includes community or comment strategy plus repurposing assets.

## Delivery hints

- A good social plan chooses channels deliberately instead of posting everywhere.
- Hooks and repurposing patterns are where platform knowledge shows up.
- Multiple post drafts are more useful than one abstract strategy paragraph.
- A strong communication pack includes where the content goes after the first post, not just the first post itself.
- Concrete stand-ins usually beat placeholder-heavy drafts when the user is asking for a launch package rather than a blank template.
