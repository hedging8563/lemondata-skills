# Usage Notes

This reference file provides public guidance that complements `research-source-scan`.

## When this skill is most useful

- The request needs a repeatable Research workflow
- The user wants a concrete deliverable, not only high-level advice
- A lightweight LemonData fast path would reduce implementation time

## Delivery hints

- Prefer a small but credible source set
- Separate direct evidence from recommendations
- Keep the summary compact and source-backed
